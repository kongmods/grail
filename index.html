Holy Grail Mod




A mod inspired by the original Holy Grail challenge

Collect all noteworthy items — the grail will keep track


Transmute any noteworthy item! The grail will keep it...            




Entirely Softcoded — Full release will be open source



     — Added 'Holy Grail' to Akara's shop

     — The Cow King can be killed infinitely

     — Enabled Ladder-only items and cube recipes

     — Enabled old patch 'Azurewrath' (there are two to find now)



The mod has been created to be vanilla-compatible, however:

     ¤ Diablo II Lord of Destruction saves only, already found items can be added

     ¤ If Cow King has already been killed, then you cannot open the portal



These items are not counted towards the Holy Grail challenge:

     » Annihilus

     » Hellfire Torch

     » Runewords

Tested on LoD 1.12, 1.13c/d

Demo — There may be bugs



Requirements:

     — Diablo II Lord of Destruction v1.12 — 1.13

     — Holy Grail patch     [download]


Installation is very simple — the mod consists of only one file "patch_d2.mpq"

     1— Find your Diablo II installation folder

     2— Inside the folder you will find a file called "patch_d2.mpq", make a backup

     3— Place the modded patch into your folder

     4— Launch the game! Akara should now be selling the Holy Grail



Likewise, uninstalling the mod is just as simple:

     ¤ Note that you must remove 'Holy Grail' item from your character

     1— Overwrite the modded patch file with your backup




Demo available for download

I suggest making a backup of your character if you intend to use an existing savefile.
As an extra feature, the demo release has a 'hidden' cube recipe: minor health potion by itself in the cube will give the player a charm with a skill that force-drops items, use it anywhere outside town.

This will not be featured in the full release
